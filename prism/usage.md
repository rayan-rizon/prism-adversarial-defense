# Camber CLI Usage

## Main

```
$ camber --help
A command line interface for interacting with the Camber platform.

Usage:
  camber [command]

Available Commands:
  login       Login to Camber platform
  app         Camber application manager
  engine      Camber engine manager
  help        Help about any command
  version     Print the version information

Flags:
      --api-key string   API key for authentication (also set via CAMBER_API_KEY env var)
  -h, --help             help for camber
  -o, --output string    Output format (table, json, yaml) (default "table")

Use "camber [command] --help" for more information about a command.
```

## Login

```
$ camber login
Authenticate to Camber platform.

Usage:
  camber login
```
